# vitaminAIChatbot

🔗 **Live Demo:**  https://m-a-h-i-s-h-a.github.io/vitdoc_without_llm/

---

## 📌 Overview

vitaminAIChatbot is a basic web application that provides responses based on predefined logic without using any Large Language Model (LLM). It demonstrates a rule-based approach to handling user inputs.

---

## 🚀 Features

* Rule-based response system
* Lightweight and fast
* Simple user interface
* No external API dependency

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How to Run Locally

```bash
git clone https://github.com/your-username/vitaminAIChatbot.git
cd vitaminAIChatbot
open index.html
```

---

## 📊 Comparison

Compared to the LLM-based version:

* Faster execution (no API calls)
* Limited intelligence
* Fixed responses (no dynamic generation)

---

## 📎 Notes

This is the **non-LLM version** of the project.
An advanced LLM-based version is implemented separately using Gemini API.
